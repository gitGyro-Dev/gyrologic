# Gyro Logic

## What is this?

Gyro Logic is a new way to understand reality.

---

## The idea

Reality is not fixed.

It is built from:

* how you observe
* what stays stable

When stability fails, you get:

* confusion
* uncertainty
* unknown states

This is called **Void**.
